# SOCKET SERVER

## Requirements

- Node.js (v16 or higher recommended)
- npm (Node Package Manager)
- TypeScript

## Instructions (English)

1. **Install dependencies**
    ```bash
    npm install
    ```

2. **Build the project**
    ```bash
    tsc -w
    ```

3. **Start the server**
    ```bash
    node dist/
    # or for auto-reload:
    nodemon dist/
    ```
